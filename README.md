### 一、学情反馈
* 概论

对直播时长，回放时长和作业分数三个维度进行反馈
* 目录结构

feedback.py是主要文件，如果需要修改可以下载，依赖的库有numpy,pandas,xlrd,xlwt,openpyxl
* 使用说明

在dist文件夹中下载feedback.exe，双击打开即可使用
1. 确保学生基本信息表与此文件在同一目录下
2. 按照提示输入信息
3. 完成后打开feedback.xlsx，反馈在第73列
